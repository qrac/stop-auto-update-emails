== Description ==

Add the function to stop automatic update emails to WordPress.

== Detail ==

* WordPress Core (Off by default)
* WordPress Themes (On by default)
* WordPress Plugins (On by default)

Activating the plugin will stop the automatic theme and plugin update emails by default.

== Frequently Asked Questions ==

Yes. You can toggle the respective email delivery on or off in the settings.
